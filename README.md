# Diabetes Prediction System

## Overview
Machine learning model to predict diabetes based on patient health data.

## Tech Stack
- Python
- Scikit-learn
- Pandas, NumPy, Matplotlib

## Features
- Data preprocessing
- Feature engineering
- Model training & evaluation

## Results
Achieved 77% accuracy using classification algorithms.

## How to Run
1. Clone repo
2. Install requirements
3. Run main file
